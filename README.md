## 📌 Deskripsi Proyek

Aplikasi **Profil Mahasiswa** adalah aplikasi mobile berbasis Flutter yang dirancang untuk menampilkan data diri mahasiswa secara modern, minimalis, dan interaktif.

Proyek ini dibuat untuk memenuhi tugas praktikum dengan ketentuan:

* Membuat aplikasi 2 halaman
* Menampilkan profil lengkap mahasiswa
* Menggunakan navigasi antar halaman
* Memiliki UI menarik
* Menggunakan minimal 5 widget layout

---

## 🎯 Tujuan Aplikasi

Aplikasi ini bertujuan untuk:

* Menampilkan informasi profil mahasiswa secara digital
* Memberikan pengalaman UI modern dengan desain dark mode
* Melatih penggunaan Flutter widget layout
* Mengimplementasikan navigasi halaman menggunakan `Navigator`
* Menggunakan animasi sederhana agar aplikasi lebih interaktif

## 🚀 Cara Menjalankan Project

### 1. Clone Repository

```bash
git clone https://github.com/username/profil-mahasiswa-flutter.git
```

### 2. Masuk ke Folder Project

```bash
cd profil-mahasiswa-flutter
```

### 3. Install Dependencies

```bash
flutter pub get
```

### 4. Jalankan Aplikasi

```bash
flutter run
```
